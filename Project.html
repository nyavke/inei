04  ЗАЧЕМ ОН НУЖЕН

Кому и как может помочь.

Жесты — один из самых естественных способов общаться с машиной. Никаких пультов, проводов и кнопок — достаточно показать руку.

01

Люди с ограниченными возможностями

Управление помогающим роботом или коляской без необходимости держать пульт и нажимать мелкие кнопки — достаточно показать руку перед камерой.

02

Медицина и стерильные зоны

Хирург или лаборант может управлять приборами и роботизированными ассистентами, не прикасаясь к ним руками — это сохраняет стерильность.

03

Образование

Наглядный кейс «нейросеть → код → железо» для уроков робототехники: показывает, как Python, ML и IoT работают вместе.

04

Промышленность

На производстве оператор может управлять манипулятором или конвейером с расстояния, не отрываясь от ручной работы и не снимая перчаток.

05

Дом и быт

Управление умной техникой и игрушками без поиска пульта или телефона — жест перед камерой запускает нужный сценарий.

06

Игры и развлечения

Бесконтактный контроллер для роботов-питомцев, гонок и интерактивных игр — рука превращается в джойстик.

06  ОПИШИТЕ ПРОГРАММУ

Как это работает.

Код написан на Python и запускается в Google Colab. Использует MediaPipe для распознавания, OpenCV для обработки кадров и paho-mqtt для передачи команд роботу.

— Карта жестов

stop
Открытая ладонь
Четыре пальца вверх — стоп.
forward
Указательный
Один палец вверх — вперёд.
right
Два пальца
Указательный + средний — вправо.
left
Три пальца
Указательный, средний, безымянный — влево.
back
Кулак
Все пальцы согнуты — назад.
gesture_classifier.py 
# Считаем поднятые пальцы и сопоставляем жест с командой
def recognize_gesture(landmarks):
    def is_extended(tip, pip):
        return landmarks[tip].y < landmarks[pip].y

    fingers = [
        is_extended(8,  6),   # указательный
        is_extended(12, 10),  # средний
        is_extended(16, 14),  # безымянный
        is_extended(20, 18),  # мизинец
    ]
    count = sum(fingers)

    if   count == 4: return "stop"
    elif count == 0: return "back"
    elif count == 1 and fingers[0]: return "forward"
    elif count == 2 and fingers[0] and fingers[1]: return "right"
    elif count == 3 and fingers[0] and fingers[1] and fingers[2]:
        return "left"
    return "none"

— Алгоритм по шагам

→ 01

Захват кадра

JavaScript внутри Colab запрашивает камеру браузера и каждые ~80 мс отдаёт кадр 640×480 в Python в виде base64.

→ 02

Детекция руки

MediaPipe Hands находит руку на кадре и возвращает координаты 21 ключевой точки — основания и фаланг пальцев.

→ 03

Классификация

Простая логика: палец считается «поднятым», если его кончик выше среднего сустава. По числу и комбинации поднятых пальцев определяется жест.

→ 04

Стабилизация

Команда отправляется, только если один и тот же жест зафиксирован 5 кадров подряд. Это убирает дрожание и ложные срабатывания.

→ 05

Публикация в MQTT

Клиент paho-mqtt подключается к broker.emqx.io и публикует строку команды в топик ligarobotov/darkfaerie.

→ 06

Исполнение

Робот подписан на тот же топик. Получив команду, он сразу выполняет движение — вперёд, назад, влево, вправо или стоп.
